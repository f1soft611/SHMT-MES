## Interface Documents Directory

### Security Notice

The following files have been removed from the public repository due to sensitive company information:

- ERP-MES Interface specifications
- Work Breakdown Structure (WBS)
- Requirements documents

### Access to Documents

Internal developers can access these documents through:

1. Internal document management system
2. Contact project manager
3. Company shared drive (with proper permissions)

### Security Policy

Do not commit files containing:
- Company confidential information
- Detailed database schemas
- Actual business or customer data
- Internal infrastructure diagrams

For more information, see [SECURITY.md](../../SECURITY.md)
